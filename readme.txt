A plugin show Facebook Page Reviews and Google Places Reviews on your websites in fixed position or widget.

== Description ==

This plugin will display your Google My Business and Facebook reviews trust badges on your WordPress websites in fixed position part of your theme widgets (check screenshots) or floating left or right.
Online Demo: [https://www.shounakgupte.com](https://www.shounakgupte.com/)

**Features:**

+ Always Free
+ Easy to install
+ Fixed position or floating left or right
+ Display total number of reviews
+ Open links in new window
+ Nofollow link to your Google My Business listing and Facebook page
+ Auto fetch data from Google and Facebook
+ Supports shortcode

**Future features:**

+ Rich snippet support
+ Write a review button
+ Information storing in database
+ Review sliders
+ Minimum review ratings filter

**Shortcode:**

Facebook Page Reviews : [facebook_badge page_id="Facebook Page ID" page_name="Facebook Page Name"]
Google Places Reviews : [google_badge place_id="Google Place ID" place_name="Goo

**NOTE:** Go to Settings -> Total Reviews -> Shortcode Helper for more information

== Installation ==

1. Unzip the downloaded plugin archive.
2. Upload the inner 'total-reviews-widget' directory to the '/wp-content/plugins/' directory on your web server.
3. Activate the plugin through the 'Plugins' menu in WordPress.
